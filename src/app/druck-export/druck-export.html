Der Genogramm Designer verfügt nicht nur über zahlreiche, in keinem anderen vergleichbaren Programm enthaltenen Funktionen zur individuellen Anpassung eines Genogramms, sondern unterstützt den Anwender auch mit einer Vielzahl von Druck- und Export-Funktionen bei der Weitergabe von Genogrammen.

Jedes Genogramm kann in folgende Formate konvertiert werden:

  • BMP -> Bitmap
  • GIF -> Graphics Interchange Format
  • JPG -> Joint Photographic Experts Group
  • PNG -> Portable Network Graphics
  • PDF -> Portable Document Format
  • Zwischenablage -> Das Genogramm wird als hoch-aufgelöste Grafik im Speicher abgelegt und kann in andere Anwendungen, z.B. Microsoft Word, als Grafik eingefügt werden.
