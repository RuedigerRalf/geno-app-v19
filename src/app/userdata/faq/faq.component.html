Wie schalte ich den Genogramm Designers mit meiner Lizenz frei?
  1. Erstellen Sie ein Verzeichnis auf Ihrem PC oder auf einem USB Stick.
  2. Speichern Sie zip Datei in dem erstellten Verzeichnis und entpacken die Datei dort.
  3. Starten Sie den Genogramm Designer und Klicken im Hauptmenü in der Rubrik 'Hilfe' auf 'Lizenzierung'. Es öffnet sich ein Eingabedialog.
  4. Zur Eingabe Ihrer Daten stehen jetzt zwei Optionen zur Auswahl:
    1. Klicken Sie im Menü auf die Schaltfläche 'Manuelle Eingabe'.
      Die Felder des Eingabedialogs werden freigegeben und Sie können Ihre Daten - siehe pdf Datei - manuell eingeben.
    2. Klicken Sie im Menü auf die Schaltfläche 'Lizenz importieren'.
      Navigieren Sie zu dem von Ihnen erstellten Verzeichnis, in dem sich die Datei license.lic befindet. Wählen Sie diese Datei aus. Ihre Daten werden automatisch übernommen.
